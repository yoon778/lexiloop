# 개인 베타 체크리스트

상태: 확정

기간: 1개월

목적: 학습 성과 증명이 아닌 기능·데이터·사용성 QA

## 시작 전

- [ ] 실제 Android 기기 1대 이상 준비
- [ ] Android 8/API 26 에뮬레이터 기본 동작 확인
- [ ] Android 16/API 36 기기 또는 에뮬레이터 확인
- [ ] `local.properties`에 Gemini 키 설정
- [ ] Gemini API 전용 키 제한 확인
- [ ] Git 추적 파일과 APK 로그에 키 없음 확인
- [ ] 기본 오프라인 단어장 300개 포함
- [ ] JSON 백업 1회 생성 후 별도 위치 보관
- [ ] 알림 권한 거부 상태에서도 앱 사용 가능
- [ ] 미국식 TTS 음성 미설치 상태 처리 확인

## 최초 사용

- [ ] 학습 목적 입력·난이도·하루 학습량 저장
- [ ] 빈 목적·과도하게 긴 목적 입력 차단
- [ ] AI 분석 성공·재시도·취소
- [ ] 네트워크 끊김 시 기본 단어장 선택 가능
- [ ] 50개 배치 일부 오류 시 전체 미저장
- [ ] 앱 재시작 후 온보딩 위치 복구
- [ ] 진단 20개 완료 전·후 복구

## 신규 학습

- [ ] 오늘 목록 수와 실제 세션 목표 일치
- [ ] 안다 경로에서 객관식 생략
- [ ] 안다 검증 실패 시 일반 경로 복귀
- [ ] 철자 대소문자·공백만 무시
- [ ] 철자 2회 실패 후 힌트 표시
- [ ] 문장 빈칸 정답 형태 판정
- [ ] 정답 0.7초 후 자동 진행
- [ ] 오답은 사용자가 다음 선택
- [ ] 나중에 다시가 당일 확인 수에 포함
- [ ] 단어장 제외 후 다시 출제되지 않음
- [ ] 세션 도중 종료 후 정확한 단계 복구
- [ ] 날짜 변경 시 미완료 신규 항목 대기열 복귀

## 복습

- [ ] 당일·1·3·7·14·30일 간격 전이
- [ ] 홀수 간격 철자, 짝수 간격 문장 시작
- [ ] 일부 오답은 이전 간격
- [ ] 완전 실패는 1일 단계
- [ ] 30일 통과 후 장기 기억 처리
- [ ] 예정 복습 21개부터 신규 잠금
- [ ] 20개 이하 처리 후 신규 잠금 해제
- [ ] 오래 밀린 항목 우선
- [ ] 이미 시작한 신규 세션은 잠금 발생 후에도 유지

## 데이터

- [ ] 앱 강제 종료 후 Room·DataStore 복구
- [ ] JSON 내보내기 성공
- [ ] 정상 JSON 가져오기 후 데이터 일치
- [ ] 손상 JSON 가져오기 거부와 기존 데이터 유지
- [ ] 다른 스키마 버전 오류 안내
- [ ] 학습 기록만 초기화
- [ ] 추천 단어장만 재생성
- [ ] 제외 단어 복원 시 기존 진도 유지
- [ ] 전체 삭제 전 확인, 삭제 후 온보딩 이동
- [ ] 데이터베이스 destructive migration 없음

- [ ] Android 13 이상 알림 권한 거부·허용

- [ ] 재부팅 후 알림 재예약
- [ ] 시간대·기기 시각 변경 후 재예약
- [ ] TTS 자동 재생·다시 듣기
- [ ] TTS 실패가 학습 진행을 막지 않음
- [ ] 오프라인에서 저장된 학습·복습 완료

## 중단 기준

다음 발생 시 신규 학습을 중단하고 먼저 수정

- 데이터 손실 또는 진도 역행
- 같은 단어 무한 반복
- 복습 일정 대량 오염
- API 키 노출
- 백업이 기존 데이터를 부분 삭제
- 앱 시작 불가 또는 반복 충돌
