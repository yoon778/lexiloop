# UI 기획

## 방향

모토: 깔끔함

- 한 화면 한 행동
- 장식보다 여백과 타이포그래피
- 핵심 단어 크게
- 보조 정보는 약하게
- 주요 강조색 1개
- 과한 애니메이션과 캐릭터 없음

## 접근성

- Android 시스템 밝은·어두운 테마 반영
- 시스템 글자 크기 반영
- 충분한 터치 영역
- 정답과 오답을 색상만으로 구분하지 않음
- 목표 뜻은 색상과 `주요 뜻` 표기를 함께 사용
- 보조 뜻은 흐리게 표시하되 읽을 수 있는 명암 유지
- TalkBack용 의미 있는 설명 제공

## 주요 화면

### 온보딩

- 학습 목적 자연어 입력
- 난이도 자가 선택
- 하루 신규 단어 수
- 알림 시각
- AI 분석 결과 확인 팝업

### 수준 진단

- 단어 1개 중심
- `안다`, `헷갈린다`, `처음 본다`
- 20개 진행 상태 최소 표시

### 홈

- 오늘 복습 수
- 오늘 신규 목표 수
- 연속 학습일
- 누적 학습 단어 수
- 복습 시작 버튼
- 신규 학습 버튼
- 복습 21개 이상일 때 신규 잠금 이유 표시

### 오늘의 단어 목록

- 20개 빠른 훑기
- 영어 표현과 목표 뜻만 우선 표시
- 상세 내용 과밀 금지

### 개별 단어 카드

- 영어 표현
- 발음기호와 다시 듣기
- 주요 뜻 강조
- 보조 뜻 흐리게 표시
- 품사
- 짧은 예문과 해석
- 하단 고정 선택: `안다`, `헷갈린다`, `처음 본다`

### 객관식

- 질문 1개
- 선택지 4개
- 정답 위치 무작위
- 선택 즉시 피드백

### 철자·문장 입력

- 입력 대상과 힌트 명확히 구분
- 키보드 등장 시 버튼이 가려지지 않음
- 첫 글자, 글자 수, 뜻 힌트
- 공백·대소문자 무시 안내는 처음에만 짧게 표시

### 오답 확인

- 틀린 입력
- 정확한 정답
- 목표 뜻
- 예문과 해석
- 자동 발음
- 사용자가 `다음` 선택

### 결과

- 완료 메시지
- 오늘 학습한 신규 수
- 오늘 처리한 복습 수
- 과도한 통계 없음

### 설정

- 하루 신규 단어 수
- 알림 시각과 켜기·끄기
- 관심사·난이도 변경
- 추천 대기열 재생성
- 제외 단어 복원
- 데이터 내보내기·가져오기
- 학습 기록 초기화
- 전체 데이터 삭제
- 오류 로그 내보내기

## 피드백 속도

- 정답: 약 0.7초 후 자동 다음 문제
- 오답: 사용자가 내용을 확인한 뒤 직접 다음 문제
- 불필요한 전환 애니메이션 제외
